## 🧠 Solution Format

For every problem, I try to include:

1. **Problem summary (in my own words)**  
2. **Intuition — the key idea behind the solution**  
3. **Step-by-step approach**  
4. **Time and space complexity analysis**  
5. **Final optimized implementation**

The goal is to **understand patterns**, not memorize solutions.
